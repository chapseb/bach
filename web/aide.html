Utilisation de l'outil de recherche archivistique Bach

 

L’annuaire d’entreprise vous donne accès aux coordonnées de tous les acteursous sites confondus.

Il contient également des numéros spéciaux ou de ressources, telles que salles de réunion, visio,….

 

L’annuaire d’entreprise vous donne accès à des informations sur les équipes de l’entreprise (responsable, organigramme,…) et sur les missions exercées dans l’entreprise (correspondants, concepteurs, élus,…).

 

Pour les contacts de type personnes, vous avez à votre disposition les fonctionnalités de consultation de l’agenda et de mailto.

 

 

Mise à jour de l’annuaire d’entreprise

 

La mise à jour de la majorité des données de contacts gérées dans cet annuaire (téléphone, bureau,…) se fait de manière automatisée en liaison avec l’outil JIRA et les demandes de mouvements de personnel.

 

Bouton « Connexion » :

 

- Le manager gère la présentation de son équipe :

  . description synthétique et périmètre fonctionnel

  . organisation en pôles de son équipe : responsable et description du pôle

 

- Les administrateurs fonctionnels gèrent les personnes affectées aux différentes missions exercées dans l’entreprise (correspondants, concepteurs, élus,…).

 

 

Mode déconnecté

 

Une version restreinte de l’annuaire est distribuée sur les portables ; elle est accessible via un raccourci sur le bureau.

 

 

Contact

 

Vous disposez d’un bouton « Rapporter une erreur » qui vous permet de nous signaler toute erreur que vous pourriez rencontrer dans les données reprises dans cet annuaire.
